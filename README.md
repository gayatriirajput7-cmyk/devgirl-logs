## 🌸 About This Project
This repository holds **The Dev Diary**, a responsive, aesthetic blog website built to showcase my 5-day progression in learning frontend web development. 

Instead of a generic practice page, I wanted to build something real that tracks my actual milestones—from writing my first `<div>` on Day 1 to mastering **CSS Grid and Flexbox** today!

## 🚀 What I Learned (The Glow-Up)

* **Day 01: HTML Foundations:** Understood DOM structure, semantic tags, and why closing tags matter.
* **Day 03: Structuring the Chaos:** Moved away from "div soup" to clean semantic layout using `<header>`, `<main>`, `<article>`, and `<nav>`.
* **Day 05: CSS Magic & Layouts:** 
  * Mastered **Flexbox** for alignment and navigation bars.
  * Implemented responsive **CSS Grid** (`auto-fit`, `minmax`) for the blog card layout.
  * Added custom CSS variables for a cohesive warm/pastel aesthetic.
  * Built custom hover states, CSS keyframe floating animations, and glassmorphism UI effects.
  * Integrated an inline DOM manipulation script for an interactive "Cheer Counter" button!

## 🛠️ Tech Stack
* **HTML5:** Semantic architecture
* **CSS3:** Grid, Flexbox, Keyframe Animations, Custom Variables
* **JavaScript:** DOM manipulation (Interactive cheer widget)
